# **App Name**: AegisRecovery

## Core Features:

- Site Navigation & Layout: Implements the core website structure, including a sticky header with blur effect, primary navigation links, multiple call-to-action buttons, and a responsive mobile hamburger menu.
- Dynamic Hero Section: Displays the main headline, subtext, key trust metrics (e.g., amount recovered, success rate), and prominent calls to action in a visually engaging layout.
- Solutions & Process Visualization: Showcases various recovery service cards with icons and descriptions, and presents a clear, step-by-step visual guide (timeline) for the recovery process.
- AI-Assisted Recovery Request Form: Provides a comprehensive submission form with client-side validation (using React Hook Form and Zod) and an AI tool to help users draft their recovery request message by suggesting optimal phrasing or necessary details.
- Dynamic Trust Indicators: Integrates testimonial cards with star ratings and quotes (carousel on mobile, grid on desktop) and visual imagery of expert teams and secure systems to reinforce legitimacy and trust.
- Knowledge Hub: Displays a grid of blog article cards (title, category, description) with a 'Load More' feature to establish authority and enhance SEO.

## Style Guidelines:

- Primary color: Vibrant blue (#3D92FF) for interactive elements, key information, and trust signals, conveying clarity and digital reliability.
- Background color: Deep navy-charcoal (#16171A), creating a modern and professional dark-mode base that anchors the visual design.
- Analogous accent: Light cyan (#A3E6F4), used for subtle highlights and complementary design elements, enhancing readability.
- Secondary accent: Gold (e.g., #FFC107), specifically requested for key calls to action and premium distinction.
- Highlight color: Teal (e.g., #20C997), specifically requested for distinct user interface highlights and visual separation.
- Headlines: 'Playfair' (serif) for an elegant, high-end, and fashionable feel; Body text: 'Inter' (sans-serif) for clean readability and a modern aesthetic.
- Use clear, professional icons for service cards and circular step indicators, prioritizing simplicity and modern outline or subtle gradient styles.
- Modern and clean layout featuring rounded cards, soft shadows for depth, thin borders for refinement, and strategic use of grid for adaptive content presentation across devices.
- Subtle and sophisticated animations, implemented with Framer Motion, for scroll-triggered effects, smooth hover states, and gentle pulsing on key call-to-action buttons to enhance user engagement.
